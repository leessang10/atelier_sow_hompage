# 홈페이지 개선 제안서

건축 사무소의 아이덴티티인 "본질", "정성", "공간감"을 웹사이트 경험으로 확장하는 데 초점을 맞춘 개선 계획입니다.

## 1. About (회사 소개)
**현재 상태:** 텍스트와 이미지의 단순한 2단 그리드, 하단 인물 소개 배치.
**제안 사항:**
*   **비대칭 레이아웃 (Asymmetrical Layout):** 정직한 그리드 대신 텍스트와 이미지를 살짝 겹치거나, 여백을 과감하게 활용하여 건축적인 긴장감과 공간감을 부여합니다.
*   **타이포그래피 강조:** 'Atelier SOW'의 의미(심다 + Sound of Wise)를 설명하는 부분을 큰 따옴표나 독특한 타이포그래피 배치로 시각적 포인트로 만듭니다.
*   **스크롤 스토리텔링:** 스크롤에 따라 "철학 -> 인물 -> 비전" 순서로 콘텐츠가 부드럽게 떠오르며 이어지도록 애니메이션을 강화합니다.

## 2. Projects (프로젝트 목록)
**현재 상태:** 단순한 그리드 형태의 프로젝트 나열.
**제안 사항:**
*   **Masonry 그리드 또는 가변형 레이아웃:** 모든 이미지가 동일한 크기인 바둑판식 배열보다는, 프로젝트의 특성에 따라 어떤 것은 크게(가로 2칸), 어떤 것은 세로로 길게 배치하여 리듬감을 줍니다.
*   **카테고리 필터:** 주거(Residential), 상업(Commercial), 인테리어(Interior) 등 탭 메뉴를 추가하여 방문자가 원하는 포트폴리오를 쉽게 찾을 수 있게 합니다.
*   **호버 인터랙션:** 마우스를 올렸을 때 이미지가 살짝 확대되거나(Zoom-in), 흑백이 컬러로 변하는 등 감각적인 마이크로 인터랙션을 추가합니다.

## 3. Press (보도자료)
**현재 상태:** 비어 있는 헤더 타이틀, 단순 목록.
**제안 사항:**
*   **타임라인 디자인:** 연도별/날짜별 흐름을 알 수 있도록 수직 타임라인 형태로 디자인하여 회사의 성장 과정을 보여줍니다.
*   **카드 디자인 개선:** 기사 썸네일과 제목, 요약문이 잡지 레이아웃처럼 보이도록 정돈된 카드 스타일을 적용합니다.
*   **헤더 보강:** 비어 있는 타이틀 영역에 "Media Coverage" 또는 "Our Stories" 같은 문구를 넣어 페이지의 목적을 명확히 합니다.

## 4. Process (진행 과정)
**현재 상태:** 단순 박스 형태의 나열.
**제안 사항:**
*   **Flow 연결성 시각화:** 각 단계(Step)가 독립된 박스가 아니라, 선(Line)으로 연결되어 하나의 흐름처럼 보이도록 디자인합니다. 건축 프로세스의 유기적인 연결성을 강조합니다.
*   **아코디언/탭 UI:** 전체 과정을 한눈에 보여주되, 각 단계를 클릭하면 상세 내용(필요 서류, 소요 기간 등)이 펼쳐지는 방식으로 정보의 강약을 조절합니다.
*   **일러스트/아이콘:** 각 단계를 상징하는 미니멀한 라인 아이콘을 사용하여 직관성을 높입니다.

## 5. Contact (문의하기)
**현재 상태:** 폼과 지도의 수직 배치.
**제안 사항:**
*   **Split Screen (화면 분할):** 데스크톱 기준 좌측에는 연락처 정보와 지도, 우측에는 문의 폼을 고정 배치하여 정보를 한눈에 볼 수 있게 합니다.
*   **입력 폼 디자인:** 입력 필드의 테두리만 남기는 미니멀한 스타일(Underline Style)을 적용하여 깔끔함을 유지합니다.
*   **지도 스타일:** 구글 맵의 스타일을 흑백(Grayscale) 또는 브랜드 컬러톤으로 커스텀하여 사이트 전체 무드와 이질감이 없도록 합니다.

## 6. React Bits 라이브러리 활용 제안 (https://reactbits.dev)
제안된 개선 사항을 구현하기 위해 다음과 같은 React Bits 컴포넌트 도입을 고려합니다.

### About (회사 소개)
*   **Scroll Reveal / Scroll Float:** 텍스트가 스크롤에 따라 우아하게 나타나는 효과. 철학 설명 시 적합.
*   **Scroll Stack:** 섹션들이 겹쳐지며 넘어가는 효과. 깊이감 부여.

### Projects (프로젝트)
*   **Masonry:** 다양한 크기의 이미지를 자연스럽게 배치하는 레이아웃.
*   **Glare Hover / Spotlight Card:** 마우스 호버 시 빛 반사나 조명 효과로 고급스러운 느낌 연출.

### Press (보도자료)
*   **Tilted Card:** 마우스 움직임에 따라 3D로 기울어지는 카드 효과.
*   **Decay Card:** 이미지가 흩어지는 듯한 독특한 노이즈 효과.

### Process (진행 과정)
*   **Stepper:** 단계별 진행 상황을 시각화.
*   **Magnet Lines:** 마우스에 반응하는 선으로 유기적인 연결성 표현.

### Contact (문의하기)
*   **Iridescence / Waves:** 배경에 은은한 물결이나 오로라 효과를 주어 감성적인 분위기 연출.
